# XYZ Reader

![Cover image](images/cover.png)

<a href='https://play.google.com/store/apps/details?id=io.github.artenes.xyzreader&pcampaignid=MKT-Other-global-all-co-prtnr-py-PartBadge-Mar2515-1'><img alt='Get it on Google Play' src='https://play.google.com/intl/en_us/badges/images/generic/en_badge_web_generic.png' width="200px" /></a>

Android application to browse a short list of articles from a remote endpoint. This application was an exercise to improve the UI from [Udacity's XYZ Reader](https://github.com/udacity/xyz-reader-starter-code).

# Features

- Browse articles
- Read articles
- Share articles (shares nothing)
